# Kytsunia

Telegram бот з характером для приватних чатів.

## Розробка

Потрібен Node 22+ і pnpm.

```bash
pnpm install
cp .env.example .env
# заповни .env
pnpm dev
```

## Vision (фото)

Кицюня вміє бачити фото. Працює коли модель — Sonnet (або інша з vision):
картинка з тригер-повідомлення, картинки в reply-таргеті, та останні фото з
історії чату підтягуються разом з текстовим контекстом. Альбоми Telegram
розпізнаються як одне повідомлення з кількома фото.

Налаштування (env):

- `KYTSUNIA_VISION_ENABLED` (default `true`) — фіче-флаг
- `KYTSUNIA_MAX_PHOTOS_TOTAL` (8) — cap на кількість фото в одному запиті
- `KYTSUNIA_MAX_PHOTOS_PER_ALBUM` (5) — cap на один альбом у контексті
- `KYTSUNIA_VISION_ALBUM_DEBOUNCE_MS` (1500) — затримка для збору сіблінгів альбому

Скачані фото кешуються в SQLite за `file_unique_id` — повторні відповіді в
тому ж треді не качають Telegram заново.

## Privacy

`/забудь мене` (opt-out) стосується **тільки персональних профілів** —
Кицюня перестає вивчати твій стиль. Повідомлення в контексті чату
(включно з фото) залишаються видимими для бота, бо без них він не зможе
підтримати розмову.
